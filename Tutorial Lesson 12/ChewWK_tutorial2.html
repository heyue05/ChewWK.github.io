    7. Main Language Function Role in Web Development
    HTML
    (HyperText Markup Language)    		 Defines the structure and content of a webpage (e.g., headings, paragraphs, images, links) Provides the structure of the website
    CSS
    (Cascading Style Sheets)    		 Controls the style and appearance (e.g., colors, fonts, layout) Provides the presentation or look of the site
    JavaScript Adds interactivity and dynamic behavior (e.g., click events, form validation) Provides the behavior and interactivity
