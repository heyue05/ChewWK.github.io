  4. State TWO (2) differences between HTTP and HTTPS.

    5. HTTP HTTPS
    No authentication. Uses SSL certificates to verify website identity.
    There is no lock icon in the browser address bar. A lock icon is displayed in the browser address bar, indicating identity authentication.


  5. State TWO (2) differences between static website and dynamic website.

    6. Static Website Dynamic Website
    Web pages are prebuilt source code. Web pages are not prebuilt source code.
    Resources load quickly, suitable for displaying information that is not updated frequently. It has high requirements for the server, and may be slightly slower when generating content, but it supports personalized content display.
